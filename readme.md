#  简介
AngularJS 是一个js客户端mvc框架。主要用来开发单页面应用。非常适合编写有大量CRUD操作的，具有Ajax风格的富客户端应用。
## 特点
创新的模板(templating)系统、数据双向绑定、友好的开发过程、可靠的工程实践。依赖注入和可测试性
## MVC
angjular控制器、作用域和视图的协同工作
## 两个关键点
模块和服务。 原理是angular框架的依赖注入系统
